<head> </head> element

The <head> element is a container for all the head elements. Elements inside <head> can include scripts, instruct the browser where to find style sheets, provide meta information, and more.

The following tags can be added to the head section: <title>, <style>, <meta>, <link>, <script>, <noscript>, and <base>.

  • <title>
  • <style>
  • <base>
  • <meta>
  • <link>
  • <script>

Defines the title of the document and is required in all HTML documents. The <title> element:

  • defines a title in the browser toolbar
  • provides a title for the page when it is added to favorites
  • displays a title for the page in search-engine results

Memorize This! Used to define style information for an HTML document. Inside the <style> element you specify how HTML elements should render in a browser.

Try This Above! 

<style type="text/css">
    body {background-color:yellow;}
    p {color:blue;}
</style>

Specifies the base URL/target for all relative URLs in a page.

<base href="http://www.w3schools.com/images/" target="_blank">

Provides metadata about the HTML document. Metadata will not be displayed on the page, but will be machine parsable. The metadata can be used by browsers (how to display content or reload page), search engines (keywords), or other web services. <meta> tags always go inside the <head> element.

Meta elements are typically used to specify page description, keywords, author of the document, last modified, and other metadata.

Define a description of your web page
<meta name="description" content="Free Web tutorials on HTML and CSS">
Define keywords for search engines
<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">
Define the author of a page
<meta name="author" content="Hege Refsnes">
Refresh document every 30 seconds
<meta http-equiv="refresh" content="30">

Memorize This! Defines the relationship between a document and an external resource and is most used to link to style sheets. 

<link rel="stylesheet" type="text/css" href="mystyle.css">

Used to define a client-side script, such as a JavaScript. Also with <noscript>. We will go over this in Javascript class.

<body> </body> element

The <body> element defines the body of the HTML document. All inline and block elements belong between the <body></body> tags.
