<h1> to <h6> tags

HTML headings are defined with the <h1> to <h6> tags.

Try This Above! Insert after <body> tag. 

<h1>My First Header</h1>
<h2>Rename me!</h2>
<h3>Me too!</h3>
<h4>I'm the smallest.</h4>
<h5>No, I am.</h5>
<h6>Actually, I think I'm the smallest.</h6>

Additional Info

Heading tags are block level elements. Block level elements normally start (and end) with a new line when displayed in a browser. When you insert the code above, notice how the headings stack on top of each other, instead of lining up from right to left. Each block takes up a line.

Proper Syntax

  • <h1> Heading 1 </h1>
  • <h2> Heading 2 </h2>
  • <h3> Heading 3 </h3>
  • <h4> Heading 4 </h4>
  • <h5> Heading 5 </h5>
  • <h6> Heading 6 </h6>

Try This Above! Insert after <body> tag or after any <h1> to <h6> tags.
