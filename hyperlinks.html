the <a> anchor tag

The HTML <a> tag defines a hyperlink.

Try This Above! Insert after <body> tag. 

<a href="http://www.w3schools.com/">Visit W3Schools</a>

Tag Information

A hyperlink (or link) is a word, group of words, or image that you can click on to jump to another document. The most important attribute of the <a> element is the href attribute, which indicates the linkâ€™s destination.

By default, links will appear as follows in all browsers:

  • An unvisited link is underlined and blue
  • A visited link is underlined and purple
  • An active link is underlined and red

Proper Syntax

<a href="url">Link Text</a>

<p> ... </p> element

HTML paragraphs are defined with the <p> tags.

Try This Above! Insert after <body> tag or after any <h1> to <h6> tags.

<p>My name is (insert name here). I studied (insert major here) at (insert school here). I am learning how to code in HTML and CSS. I'll write 3 more sentences which includes my hobbies, favorite meal, and why I'm learning web design.</p>

Tag Information

Paragraph tags are also block level elements.

Proper Syntax

<p> Insert Paragraph Here </p>

formatting tags

HTML uses formatting tags for formatting output. Use sparingly and appropriately.

Try This Above! Insert after <body> tag.

<p>I would like to <em>emphasize</em> the importance of <strong>being focused and on task</strong>. <del>Hopefully I'm not wrong about this.</del> If you can <del>finish</del> <ins>complete</ins> your task <mark>efficiently and with high quality</mark>, you prevent redoing work for any noticable incorrections. Staying focused demands willingly<sup>1</sup> and consciously<sup>2</sup> limiting your distractions. Being on task promotes punctuality and reliability. </p>
<p><sup>1</sup> You need a <strong>resolve</strong>.</p>
<p><sup>2</sup> You need to be <strong>alert</strong> and <strong>identify</strong> your distractions.</p>

Tag Information

Formatting tags are inline level elements. Inline elements are normally displayed without starting a new line. There are also Computer Output tags and tags for Citations, Quotations, and Definitions. There is no longer a need to use the following formatting tags: <b>, <i>.

Proper Syntax

Formatting Tags

  • <strong> Important Text </strong>
  • <em> Emphasized Text </em>
  • <small> Smaller Text </small>
  • <sub> Subscripted Text </sub>
  • <sup> Superscripted Text </sup>
  • <ins> Inserted Text </ins>
  • <del> Deleted Text </del>
  • <mark> Highlighted Text </mark>

Computer Output Tags

  • <code> Computer Code Text </code>
  • <kbd> Keyboard Text </kbd>
  • <samp> Sample Computer Code </samp>
  • <var> Variable Text </var>
  • <pre> Preformatted Text - Block Element </pre>

Citation, Quotations, Definition Tags

  • <abbr> Abbreviation or Acronym </abbr>
  • <address> Contact Info - Block Element </address>
  • <bdo> Text Direction </bdo>
  • <blockquote> Quoted Text - Block Element </blockquote>
  • <q> Quote Inline Text </q>
  • <cite> Title of Work </cite>
  • <dfn> Definition Term </dfn>
